Typedef Documentation

typedef struct Color Color

an RGBA color

A Color can be initialized manually, but it's better to use init_color so errors can be caught. For the red, green, and blue components, a value of 0 indicates no expression while a value of 255 indicates full expression. For the alpha component, a value of 0 indicates full transparency, while a value of 255 indicates complete opacity.
