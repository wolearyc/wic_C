int updt_game ( Game *  target)

flips the window buffers and times game updates

This function will wait a certain amount of time before returning 1 (indicating that the game was updated and a new frame was drawn). This mechanism ensures that the fps is maintained.

Parameters
targetthe target Game; must be valid
Returns
1 if the game should be updated, 0 if the game should not be updated (for example when the window is closed by the user), < 0 on failure
