Detailed Description

An architectural superclass that should by inherited by any class that can be scaled.

Any class that inherits from Scaleable can be scaled. The object is scaled around a center, which is defined in Paintable. A scale value of (1,1) is original size, (2,2) is twice the size, (0.5,0.5) is half the size, etc.
