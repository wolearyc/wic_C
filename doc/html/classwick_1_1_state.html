Detailed Description

A game state that can be initialized, updated, and painted.

State objects are the building blocks of wick. A State represents and executes a single "game state". A State could represent the main menu or a single level of a game. The programmer should write their own classes that inherit from State. These programmer-defined States must define a destructor, initialize method, update method, and paint method.
