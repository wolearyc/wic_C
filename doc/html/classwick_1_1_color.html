Detailed Description

Defines a RGB or RGBA color.

Color also contains some common pre-defined colors, stored as static variables.

A constructor defining an RGBA (red, green, blue, alpha) color.

All component values must be in the range 0 (no expression) to 255 (complete expression). An alpha value of 255 indicates complete opacity, while an alpha value of 0 indicates total transparency.

Parameters
redthe red component (0-255)
greenthe green component (0-255)
bluethe blue component (0-255)
alphathe alpha component (0-255)
Exceptions
ParameterExceptionnon-fatal exception when the value(s) of red, green, blue, and/or alpha exceeds 255; all components that exceed 255 are set to 255
